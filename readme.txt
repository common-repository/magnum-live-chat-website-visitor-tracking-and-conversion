Magnumchat is a live chat and conversion software.  Magnumchat plugin adds Magnumchat script on website.

== Description ==

Magnumchat plugin adds Magnumchat script on website.  Magnumchat will allow you to add real-time live chat interaction, website visitor tracking, and help you increase your lead generation, sales and customer service on your blog and website.

To get started, you need a Magnumchat account. To get your free account, Sign up for a free MagnumChat account <a href="http://magnumcontact.com/livechat/wordpress-plugin-signup/">sign up here</a>.

Magnum Chat has a free trial for 14 days for wordpress plugin users.  Then, **there is no monthly charge to use the software for your first user per domain.**

**It will only take a couple of minutes to get started, and if you have any questions, we have 24/7 chat support.**

== Installation ==

1. Upload `magnumchat folder` to the `/wp-content/plugins/` directory
2. Activate the plugin magnumchat through the 'Plugins' menu in WordPress
3. You must sign up for your free Magnumchat account, Sign up for a free MagnumChat account <a href="http://magnumcontact.com/livechat/wordpress-plugin-signup/">sign up here</a>.

== Frequently asked questions ==

1- Is there any shortcode to put in any page, post or template file to add Magnumchat in the website?

Ans- No, there is no need to put any shortcode to add Magnumchat in the website.

2- How does the magnumchat plugin work?

Ans- The magnumchat plugin adds magnumchat script in the site automatically when activated.  You do need to sign up for a free Magnum Chat account.
To get your free account, Sign up for a free MagnumChat account <a href="http://magnumcontact.com/livechat/wordpress-plugin-signup/">sign up here</a>

3-  Which languages does Magnumchat support?

Ans-  Magnumchat currently supports Magnumchat Operator Software in 9 Languages. Graphical User Interface available in English, Czech and Hungarian, Spanish, Dutch, German, Slovak, Italian and Bulgarian.
